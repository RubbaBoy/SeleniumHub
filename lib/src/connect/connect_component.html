Connecting

This program is the webserver you are currently on to display instances, along with a backend API that proxies selenium instances that you connect to. For inspection, this program also has a WebSocket proxy for the instances the backend API gets. Currently this only is tested on chrome, though will most likely work for other browsers, just not inspection. To connect, you must be using selenium (no shit) on the JsonWire spec, and connect via whatever remote interface your API uses from the following URI: 

{{ host }}:{{ port }}/selenium/

The following are some examples of connecting with the server:

Dart

Using the webdriver package

var driver = await createDriver(uri: Uri.parse('http://{{ host }}:{{ port }}/selenium/'), spec: WebDriverSpec.JsonWire);

Java

Using the Selenium and Chromeium Driver libraries

var driver = new RemoteWebDriver(new URL("http://{{ host }}:{{ port }}/selenium/"), new ChromeOptions());
